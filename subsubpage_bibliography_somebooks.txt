<I> I arrived to a point where the exercise of my intellect and its purposeness 
comes before my integration in the academic world, and for this reason it is more and more difficult to distinguish what 
in this exercise is work and what is not. In particular I do not make a difference, in terms of intellectual material, between 
factual academic subjects and common life. Furthermore the books that I read work sometimes as a catalyzer for my reflexion, sometimes 
connect the conceptual realm to life, informing life with the abstract concept or the concept with life. I think that they also 
tell something about me that the texts I write do not. </I>

    function searchTable(value,data){
        var filteredData = []
        for (var i=0; i<data.length; i++) {
            value = value.toLowerCase()
            value = value.replace('é','e');
            value = value.replace('è','e');
            var author = data[i].author.toLowerCase()
            author = author.replace('é','e');
            author = author.replace('è','e');
            var title = data[i].title.toLowerCase()
            title = title.replace('é','e');
            title = title.replace('è','e');
            var year = data[i].year.toLowerCase()
            var editor = data[i].editor.toLowerCase()
            editor = editor.replace('é','e');
            editor = editor.replace('è','e');
            var field = data[i].field.toLowerCase()
            var total = author + title + year + editor + field
            if (total.includes(value)){
                filteredData.push(data[i])
            }
        }

        return filteredData
    }
